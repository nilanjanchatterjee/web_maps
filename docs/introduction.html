Background

This is a database that compiles records of published studies using Global positioning based telemetry on wild animals. Each point represents a study location associated with metadata including species name, journal, year, and DOI. Initial searches were carried out in mid-2025.

Data were harmonized and standardized using reproducible workflows in R, and exported as GeoJSON for web visualization. For further queries, please reach out to xxx@xyz.com
